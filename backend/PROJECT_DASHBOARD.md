## 🎯 Project Overview

### What Is Job Genie?
An AI-powered gig worker platform connecting blue-collar workers with short-term industrial jobs across India. **Mobile-first** (Android via Capacitor), **multilingual** (11 languages), **gamified** (XP system, leaderboard), **AI-enhanced** (Gemini chatbot).

### Target Users
- 👷 **Workers**: Gig seekers (logistics, warehousing, delivery, security)
- 👔 **Admins**: Genie Partners (job posting, management)
- 🔐 **Super Admins**: Platform oversight, fraud detection

### Core Tech Stack
- **Frontend**: React 19 (JSX) + Vite 8 + React Router v7
- **Mobile**: Capacitor 8 (Android APK)
- **Backend**: Firebase (Firestore + Auth) + Gemini AI
- **Animation**: Framer Motion 12
- **3D Effects**: Three.js, OGL, post-processing shaders

### GitHub Actions Workflow
```

   └─ ESLint checks code quality
   └─ Fails on warnings

2. 🧪 TEST
   └─ Vitest runs all unit tests
   └─ Coverage reports to Codecov
   └─ Must pass before building

3. 🔍 SECURITY
```
